##使用说明

####手工导入
所有的分类独立,可以按照需求进行导入 
如果想要全部导入只需要将`Classes`中文件拉入项目中即可

###使用
导入头文件:

使用全部:
	
	#import "SICategory.h" 
	
使用`Foundation`部分:

	#import "SIFoundation.h"
	
使用`UIKit`部分

	#import "SIUIKit.h"
